
REQUIREMENTS
------------
- Drupal 7.x 
- Views 3.x module [1]
- Libraries module [2]
- Jquery Update module [3]

- Libraries:
	jquery timeliner

[1] http://drupal.org/project/views
[2] http://drupal.org/project/libraries
[3] http://drupal.org/project/update_jquery

INSTALLATION
------------
1. Copy all module files into subdirectory sites/all/modules/vertical_timeline/
   under your Drupal installation directory.

2. Obtain the libraries of jQuery timeliner project
   * You can download a version from
     https://github.com/technotarek/timeliner

3. Place the library into a new sites/all/libraries/vertical_timeline directory.
	Just unpack the downloaded files to the directory mentioned above.
   	
4. Go to administer >> modules and enable the vertical_timeline module.
   Ensure that the Views, libraries are installed and enabled.
